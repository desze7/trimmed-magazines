Any magazines of up to 50 rounds that are 3x1 in size (3 slots tall and 1 slot wide) will get reduced to 2x1, including modded ones.
<br/>
I'm not a hundred percent sure this will work with *every* modded magazine but as far as I tested it should. If some mod isn't working please let me know.
<br/>
<br/>
<br/>
!! IMPORTANT !!
<br/>
After installing or uninstalling the mod it is very likely that the magazine icons will be clipping at the top and bottom of the slot, to fix that make sure to clean the temporary files in the SPT launcher (SPT > Settings > Clean Temp Files).
